Monitoring
Monitoring is a set a measurements used to alert for deviations from a normal range. Examples of type of data:
  1. Pod logs
  2. Network flow logs
  3. Application flow logs
  4. Audit logs
Examples of metrics:
  1. Connections per second
  2. Packages per second, in bytes
  3. Application (API) requests per second
  4. CPU and memory utilization
Examples of metrics to monitor SLA:
  1. Polling of application (API)
  2. Application response codes
  3. Application response time
  4. Node availability for scale-out use cases
  5. Memory/CPU/disk/IO resources on a node
